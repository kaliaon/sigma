Qadam жүйесінің техникалық сипаттамасы және талаптары
1. Жүйеге жалпы шолу
Qadam — бұл өзін-өзі дамытуға арналған интеллектуалды, геймификацияланған жүйе. Ол пайдаланушыға денсаулық, оқу, қаржы, қарым-қатынас және менталдық даму сияқты бес бағыт бойынша жеке өсу жоспарын ұсынуға мүмкіндік береді. Жүйе мобильді интерфейсті, серверлік API мен деректер қорын және LLM негізінде жасанды интеллектін біріктіреді.
2. Функционалдық талаптар
Пайдаланушы тіркеліп, жүйеге кіру мүмкіндігі болуы керек.
Пайдаланушының профилі мен жеке деректері сақталуы тиіс.
Жүйе күнделікті, апталық және эпикалық квесттер ұсынуы қажет.
Әр квестке XP (тәжірибе) және монета беріледі.
Пайдаланушы тапсырманы орындаған соң ұпай жинап, деңгейін көтереді.
Жүйе Google Gemini API арқылы пайдаланушыға жеке даму жоспарын (роадмап) JSON форматында құра алады.
Статистика және прогресс диаграммалары көрсетіледі.
Лидерборд және жетістіктер жүйесі қолжетімді болуы тиіс.
Жергілікті хабарламалар мен еске салғыштар жіберіледі.
Қауіпсіздік шаралары: авторизация, деректерді шифрлау және валидация.
3. Функционалды емес талаптар
Жауап беру уақыты 500 мс-тен аспауы керек.
Интерфейс қарапайым және қолдануға ыңғайлы болуы тиіс.
Жүйе қазақ, орыс және ағылшын тілдерін қолдауы қажет.
Пайдаланушы деректері шифрланып сақталуы тиіс.
Архитектура модульдік, кеңейтуге ыңғайлы болуы керек.
4. Жүйе архитектурасы
Жүйе төрт негізгі қабаттан тұрады:
1. Презентация қабаты (Frontend) — React Native негізінде жасалған мобильді интерфейс.
2. Қолданбалы қабат (Backend API) — Node.js немесе Supabase арқылы REST API жүзеге асырылады.
3. Мәліметтер қабаты (Database) — PostgreSQL деректер қоры, кестелер: users, quests, user_quests, economy, checkins.
4. Жасанды интеллект қабаты (AI Integration) — Google Gemini API пайдаланушы мақсаттарына сай жоспар құрады.
